Overview

SnuggleTeX is a free and open-source Java library for converting a reasonable subset of LaTeX to standards-compliant XHTML+MathML. It is primarily intended for use by applications requiring this type of functionality, but can also be used to generate standalone web pages very easily.

Features

  • SnuggleTeX is a 100% Java library with no dependencies on any other Java libraries.
  • Converts a good subset of (media-agnostic) LaTeX into XML.
  • Supports \newcommand and \newenvironment and friends, making it easy to create custom commands and environments. (There are also lower-level Java hooks for defining new commands and environments.)
  • SnuggleTeX has optional features for inferring limited semantics in certain mathematical expressions, such as automatically fencing matched brackets and inserting "apply function" and "invisible times" operators into simple algebraic expressions.
  • Output can either be a DOM fragment or a standalone web page.
  • Web page outputs can be configured in various ways and a there are a number of preset "page types" that support the most common ways that XHTML+MathML content can be deployed on the web.
  • Outputs can be fully standalone (e.g. with all styling done inline) to enable page fragments to be imported into other systems which support XHTML+MathML.
  • Error reporting is configurable and error messages are internationalisable and provide detailed contextual information.

Download

SnuggleTeX can be downloaded from SourceForge.net, where the project is hosted.
