\section*{Web Output Samples}

For maximum portability --- and to ensure that search engines
cope with them --- these documentation pages have
been generated using the
\href[Legacy XHTML + Images Web Page Output]{outputs.html},
which do not actually contain any MathML!

To allow you to see the different MathML-based views in
action, the links below show different versions of the
``\href[Basic Math Mode Commands]{math-mode.html}'' page
generated using some of other web output page types
described in the \href[Outputs]{outputs.xml} page.

Of course, not all of these will display in all browsers.
You should look at the \href[Outputs]{outputs.xml}
and \href[Browser Requirements]{browser-requirements.xml}
pages for more information.

  \item \href[MOZILLA]{math-mode.xhtml}

  \item \href[UNIVERSAL\_STYLESHEET]{math-mode.xml}

  \item \href[MATHPLAYER\_HTML]{math-mode.htm}

  \item \href[CROSS\_BROWSER\_XML]{math-mode.cxml}

\end{itemize}
